🔍 외주팀 입력 오류 디버깅 가이드

1️⃣ 브라우저 콘솔 열기

브라우저에서 F12 키를 누르거나

우클릭 → 검사 → Console 탭 클릭

2️⃣ 외주팀 관리 페이지 접속

  1. 좌측 메뉴에서 외주팀 관리 클릭
  2. 우측 상단 신규등록 버튼 클릭
  3. 콘솔에 표시되는 메시지 확인

3️⃣ 입력 테스트

  1. 이름/상호 입력창에 "테스트" 입력
  2. 입력한 글씨가 사라지는지 확인
  3. 콘솔에 에러 메시지가 표시되는지 확인
  4. 스크린샷 찍기 (콘솔 포함)

4️⃣ 확인해야 할 콘솔 메시지

❌ Invalid staff index: ...

TypeError: Cannot read property '...' of undefined

React Warning: A component is changing an uncontrolled input...

5️⃣ 보고할 정보

  • 어떤 필드에서 문제 발생? (이름/상호, 연락처, 활동지역 등)
  • 글씨를 입력하면 즉시 사라지나요? 아니면 잠시 후 사라지나요?
  • 콘솔에 어떤 에러 메시지가 표시되나요?
  • 스크린샷 (콘솔 포함)

📱 문제 재현 방법

1. 외주팀 관리 > 신규등록 클릭
2. 이름/상호 입력창에 "테스트" 입력
3. 문제 발생 시 콘솔 확인
4. 스크린샷 공유
