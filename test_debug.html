외주팀 입력 테스트

브라우저 콘솔을 열고 외주팀 관리 > 신규등록을 클릭하세요

입력창에 텍스트를 입력할 때 콘솔 로그를 확인하세요

예상 로그:

  • === FormData 초기화 시작 ===
  • currentCategory: OUTSOURCE
  • initialData: null
  • isOutsource: true
  • 외주팀 초기화 - outsourceTypes[0]: 시공일당
  • initialSubCategory: 시공일당
  • 외주팀 최종 초기 데이터: {id, category, brandName:'', ...}
